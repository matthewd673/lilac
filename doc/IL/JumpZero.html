Overview

A JumpZero is a Jump that will jump to the target Label only when its conditional value is equal to 0.

def eql?(other)
  if not other.class == JumpZero
    return false
  end

  other = T.cast(other, JumpZero)

  cond.eql?(other.cond) and target.eql?(other.target)
end
