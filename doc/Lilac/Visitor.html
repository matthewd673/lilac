Overview

A Visitor is designed to aid in traversing lists, trees, and etc. that contain objects of many different classes or subclasses.

Constant Summary collapse

Lambda =

A Visitor::Lambda is a type-alias for a lambda called by a Visitor. It has three parameters: the calling Visitor, the object being visited, and the optional visiting context. It can have any return type.

T.type_alias do
  T.proc.params(arg0: Visitor, arg1: T.untyped, arg2: T.untyped)
   .returns(T.untyped)
end
LambdaHash =

A Visitor::LambdaHash is a type-alias for a hash of classes -> Visitor::Lambda objects. It is used by a Visitor to determine which lambda to call for a given object.

T.type_alias do
  T::Hash[T::Class[T.untyped], Lambda]
end

def visit(obj, ctx: nil)
  obj.class.ancestors.each do |a|
    unless @visit_lambdas.include?(a) then next end

    l = @visit_lambdas[a]
    unless l # to appease call below
      raise("Visitor has a NIL lambda registered for #{a}")
    end

    return l.call(self, obj, ctx)
  end

  raise("Visitor does not support #{obj.class} or its ancestors")
end
