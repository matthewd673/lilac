Overview

A SymbolTable contains a stack of scopes and helper functions to manage the stack and the symbols within it.

def lookup(key)
  @scopes.reverse_each { |s|
    symbol = s.lookup(key)
    if symbol then return symbol end
  }
  return nil
end
